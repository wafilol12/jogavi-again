
# Jogavi Static Multipage Website

Struktur website:
- index.html
- about.html
- products.html
- services.html
- projects.html
- authorized-distributor.html
- contact.html
- assets/css/style.css
- assets/js/script.js

Fitur:
- Multi-page, bukan one-page landing saja
- Navbar clickable
- Dropdown submenu Products dan Services
- Dropdown muncul saat hover di desktop
- Dropdown bisa diklik di mobile
- Burger menu responsive
- Desain light/cerah seperti mockup
- Siap upload ke GitHub, Vercel, Netlify, atau cPanel public_html

Catatan:
- Gambar masih dibuat dengan CSS placeholder agar file ringan.
- Ganti placeholder dengan foto produk, kantor, dan project asli dari client.
